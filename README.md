# 🚫 Animated 404 Error Page

A modern and responsive **404 Error Page** built with pure **HTML5** and **CSS3**.  
The page features smooth animations, floating background elements, a glassmorphism-style button, and a clean user interface.

## ✨ Features

- 🎨 Modern gradient background
- 💫 Floating animated background shapes
- ✨ Glowing and bouncing 404 animation
- 📱 Fully Responsive Design
- ⚡ Pure HTML & CSS (No JavaScript)
- 🔤 Persian (RTL) layout support
- 🖱️ Smooth hover effects
- 🌐 Google Vazirmatn Font
